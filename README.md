# Dynamic Memory and Linked List Visualization in C++

This project implements a singly linked list using dynamic memory allocation in C++. The program allows insertion of nodes at the beginning, middle, and end of the list, and provides a graphical visualization of the structure using the Windows API.

## Features

* Dynamic memory allocation using `malloc`.
* Creation and manipulation of singly linked lists.
* Insert nodes at:

  * Beginning
  * Specific position
  * End
* Interactive console menu.
* Graphical visualization of nodes and pointers.
* Automatic display of the head and last node.

## Technologies Used

* C++
* Dynamic Memory Allocation
* Pointers
* Linked Lists
* Windows API (Win32 GDI)
* Console and Graphical Programming

## Data Structure

Each node contains:

* An integer value (`dato`)
* A pointer to the next node (`seguimiento`)

## Core Concepts Demonstrated

* Dynamic memory management
* Pointer manipulation
* Linked list traversal
* Insertion operations
* Graphical representation of data structures

## User Interaction

The program allows the user to:

1. Define the initial size of the list.
2. Enter values for each node.
3. Insert additional elements.
4. Display the linked list graphically.
